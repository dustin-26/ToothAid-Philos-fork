## Quick Instructions for Tester

- Please think out loud while using the app.
- Perform each task as you normally would in clinic.
- No right or wrong answers; we are testing the app, not you.

## Core Tasks (All Features)

Mark each task: Done / Partly / Not done

1. **Login and Home (Today)**
   - Sign in to the app.
   - Open Today and identify AM/PM patients.
   - Open one patient from queue.
   - Result: ______________

2. **Children Search and Registration**
   - Search for a child by name or patient ID.
   - Add a new child record (or review registration form fields).
   - Confirm required details are clear and sufficient.
   - Result: ______________

3. **Child Profile and Visit Entry**
   - Open a child profile.
   - Add or edit a visit (symptoms/treatment/tooth details as relevant).
   - Confirm saved visit appears correctly in profile history.
   - Result: ______________

4. **Appointment and Status Management**
   - Create a new appointment (date + AM/PM).
   - Mark statuses: Attended, Cancelled, Rescheduled.
   - Try undo to Scheduled.
   - Confirm status is reflected in Today and Schedule Day.
   - Result: ______________

5. **Schedule and Quota Management**
   - Open Schedule month view and a specific day.
   - Check/update AM/PM quotas or location if needed.
   - Use filters: Follow-up / Scheduled / Completed.
   - Confirm counts and grouping make sense.
   - Result: ______________

6. **Reports and Export**
   - Open Reports page.
   - Review key charts and totals.
   - Export report (if used in clinic workflow) and confirm output is understandable.
   - Result: ______________

7. **Offline and Sync Confidence**
   - Go offline.
   - Make one meaningful change (visit, appointment, or child update).
   - Go online and sync.
   - Confirm data persists and appears correct.
   - Result: ______________
